# 페이스스왑 템플릿 이미지 가이드

## ⚠️ 중요: 같은 모델 사진이어야 합니다!

**모든 템플릿 이미지는 반드시 같은 모델(같은 사람)의 사진이어야 합니다.**
- 이유: 일관된 체형을 유지하기 위해
- 다른 사람의 사진을 섞으면 체형이 달라져서 부자연스러움
- 같은 모델의 다양한 포즈만 사용

## 필요한 템플릿 이미지 개수

### 최소 필요 개수: **1장**
- 정면 전신샷 1장만 있어도 기본 기능 동작

### 추천 개수: **2-3장**
- 정면 전신샷 1장 (필수)
- 3/4 측면 또는 완전 측면 1-2장 (선택)

### 이상적 개수: **3-5장**
- 다양한 각도와 포즈 제공
- 사용자 선택권 확대

## 필요한 템플릿 이미지 종류

### 1. 정면 전신샷 (필수 - 최소 1장)
   - 파일명 예: `front_full_body.jpg` 또는 `template_front.jpg`
   - 용도: 가장 기본적인 피팅 이미지
   - 요구사항:
     - 슬림한 여성 체형
     - 정면을 향한 전신 사진
     - 얼굴이 명확하게 보임 (페이스스왑용)
     - 깔끔한 배경 (흰색 또는 단색 배경 권장)
     - 해상도: 최소 1024x2048 이상 권장

### 2. 3/4 측면 전신샷 (추천)
   - 파일명 예: `three_quarter_view.jpg` 또는 `template_side.jpg`
   - 용도: 측면 실루엣 확인용
   - 요구사항:
     - 약 45도 각도로 회전한 포즈
     - 드레스 실루엣이 잘 보이는 각도
     - **같은 모델의 다른 포즈**

### 3. 완전 측면 전신샷 (선택)
   - 파일명 예: `side_view.jpg` 또는 `template_profile.jpg`
   - 용도: 옆모습 실루엣 확인용
   - 요구사항:
     - 90도 측면 포즈
     - 드레스 뒷면 실루엣 확인 가능
     - **같은 모델의 다른 포즈**

### 4. 약간 회전한 포즈 (선택사항)
   - 파일명 예: `slight_turn.jpg`
   - 용도: 다양한 각도 제공
   - **같은 모델의 다른 포즈**

## 템플릿 이미지 준비 가이드

### 이미지 요구사항

1. **모델**: **반드시 같은 모델(같은 사람)**의 사진만 사용
   - 모든 템플릿은 동일 인물
   - 체형 일관성 유지 필수
   
2. **체형**: 슬림한 여성 체형 (일관된 기본 체형)
   
3. **얼굴**: 
   - 얼굴이 명확하게 보여야 함 (페이스스왑용)
   - 정면 또는 약간 측면 얼굴
   - 얼굴은 나중에 사용자 얼굴로 교체됨
   
4. **포즈**:
   - 자연스러운 전신 포즈
   - 팔 위치가 일관적 (예: 양팔 내린 자세)
   - 발 위치가 일관적
   - **같은 모델의 다양한 포즈**
   
5. **배경**:
   - 깔끔한 단색 배경 (흰색, 회색, 베이지 등)
   - 또는 투명 배경 (PNG)
   - 모든 템플릿에서 배경 일관성 권장
   
6. **해상도**:
   - 최소 1024x2048 픽셀
   - 권장: 2048x4096 픽셀 이상
   
7. **형식**: JPG, PNG, WEBP

### 파일명 규칙

- 영문 소문자와 언더스코어 사용 권장
- 예: `template_front.jpg`, `template_side.jpg`, `template_three_quarter.jpg`
- 또는: `front_full_body.jpg`, `side_view.jpg`

### 템플릿 생성 방법

1. **AI 생성**: 
   - Stable Diffusion, Midjourney 등으로 슬림 여성 전신 이미지 생성
   - 프롬프트 예: "slim woman full body, front view, white background, wedding dress template, professional photography"

2. **실사 이미지**:
   - 모델 사진 사용 (권한 확인 필수)
   - 배경 제거 후 사용

3. **3D 렌더링**:
   - 3D 모델로 일관된 체형 생성

## 사용 방법

1. 템플릿 이미지를 이 디렉토리(`templates/face_swap_templates/`)에 저장
2. 파일명은 자유롭게 설정 가능 (이미지 확장자만 올바르게)
3. API 호출 시 `template_name` 파라미터로 특정 템플릿 선택 가능
4. `template_name`을 지정하지 않으면 첫 번째 템플릿(알파벳 순) 사용

## 예시 파일 구조

### 최소 구성 (1장)
```
templates/face_swap_templates/
├── README.md (이 파일)
└── front_full_body.jpg      # 정면 전신샷 (필수, 같은 모델)
```

### 추천 구성 (2-3장)
```
templates/face_swap_templates/
├── README.md (이 파일)
├── front_full_body.jpg      # 정면 전신샷 (필수, 같은 모델)
├── three_quarter_view.jpg  # 3/4 측면 (추천, 같은 모델)
└── side_view.jpg           # 완전 측면 (선택, 같은 모델)
```

### 이상적 구성 (3-5장)
```
templates/face_swap_templates/
├── README.md (이 파일)
├── front_full_body.jpg      # 정면 전신샷 (필수, 같은 모델)
├── three_quarter_view.jpg  # 3/4 측면 (추천, 같은 모델)
├── side_view.jpg           # 완전 측면 (선택, 같은 모델)
├── slight_turn.jpg         # 약간 회전 (선택, 같은 모델)
└── back_view.jpg           # 뒷모습 (선택, 같은 모델)
```

## 주의사항

- ⚠️ **모든 템플릿은 반드시 같은 모델(같은 사람)의 사진이어야 합니다**
- 모든 템플릿은 **얼굴이 포함**되어 있어야 합니다 (페이스스왑용)
- 체형은 **일관된 슬림 체형**이어야 합니다 (같은 모델이므로 자동으로 일관됨)
- 배경은 깔끔하게 처리하는 것이 좋습니다
- 이미지 품질이 낮으면 페이스스왑 결과가 저하될 수 있습니다
- 최소 1장만 있어도 동작하지만, 2-3장 있으면 사용자 선택권이 확대됩니다
